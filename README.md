# My Academic Homepage

这个目录是一个可直接部署到 GitHub Pages 的静态学术主页。

## 文件说明

- `index.html`：主页内容（个人介绍、论文、链接等）
- `style.css`：页面样式
- `assets/images/profile.jpg`：头像（你需要自行放置图片）

## 快速部署（推荐）

1. 在 GitHub 新建仓库：`aaaaaaapp.github.io`
2. 把本目录文件上传到仓库根目录
3. 在仓库 `Settings -> Pages` 中，选择 `Deploy from a branch`
4. 分支选 `main`，目录选 `/ (root)`，保存
5. 稍等片刻，访问：`https://aaaaaaapp.github.io`

## 自定义建议

- 在 `index.html` 替换你的姓名、机构、研究方向、邮箱
- 在 `Selected Publications` 补充你的真实论文与链接
- 上传头像到 `assets/images/profile.jpg`
